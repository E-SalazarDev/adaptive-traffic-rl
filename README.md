# Adaptive Traffic RL

Full Stack traffic simulation and optimization platform using SUMO, Reinforcement Learning, Spring Boot, React and PostgreSQL.

## Goal

This project simulates adaptive traffic light control using Reinforcement Learning.

It does not control real traffic lights.  
It is a portfolio project focused on simulation, AI experimentation, backend architecture, frontend dashboards and reproducible infrastructure.
